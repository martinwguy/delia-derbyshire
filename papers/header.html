Delia Derbyshire's papers

Last updated: 28 July 2012

Here is a set of 1170 photos of Delia Derbyshire's papers, going through each folder from start to finish. Click on a thumbnail for the highest-definition version of that image.

You can also download all the images in a zip file (just over 300MB).

Here is a quick index to the start of each folder in the collection and to papers regarding identified musical pieces and performances. Clicking on a thumbnail takes you to a full-resolution version of that image. In my browser, hovering over a thumbnail tells the code name of that sheet. There are also many unidentified scores and other papers here: search for "???".
